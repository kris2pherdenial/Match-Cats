# Match-Cats
My final project for Code-In-Place 2026. A simple tile-matching game with cats.

# How to Play:
- Click two tiles to reveal the cats
- If the two cats match, they stay revealed
- If not, they stay hidden and you lose 1 attempt
- Match all pairs before you run out of attempts to win

# Features: 
- My pet cats
- Randomized tile layout in every round
- Attempts counter
- Reset Button


Written in Python with tkinter library.
